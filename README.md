# This is CRUD app 'Vartotojų kontrolė'

A simple app to create, delete, update and read users from server side.

### Installation
1. run npm install
2. change '.env example' to '.env', change port to http://localhost:5000/
3. run npm start

### Dependencies
1. bcryptjs
2. cors
3. dotenv
4. express
5. joi
6. jsonwebtoken
7. morgan
8. mysql2
