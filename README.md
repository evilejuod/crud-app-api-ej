# This is CRUD app 'Vartotojų kontrolė'

A simple app to create, delete, update and read users.
This is node js app from server side.

### Installation
1. npm install
2. copy '.env example' to '.env' and add your sql db strings

### Dependencies
1. bcryptjs
2. cors
3. dotenv
4. express
5. joi
6. jsonwebtoken
7. morgan
8. mysql2
